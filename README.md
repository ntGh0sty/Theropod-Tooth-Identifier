# Theropod-Tooth-Identifier
A repository for my CS471: Intro to AI final project, which houses the components for a model that will be trained to classify a given fossilized tooth sample to its probable species. Any components, including datasets and code, might be included at any given time. Due to the nature of the datasets that this model works off of, they will be cited at the bottom of this file, however their content validity is subject to change as new research becomes publicly available.

## Purpose
This project is to provide a bit more accessiblility for those interested in and/or committed to paleontology, as it should hopefully help users learn about what I consider to be underrepresented theropod dinosaurs! Virtually everyone knows about *T.rex* and *Spinosaurus*, but not many know about *Acrocanthosaurus* (an early large theropod dinosaur that likely preceeded the *T.rex*) or *Megalosaurus* (the first dinosaur officially described), and I want to showcase these dinosaurs in a way that suits my final project's requirements.
